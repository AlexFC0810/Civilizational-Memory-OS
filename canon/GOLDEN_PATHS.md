# Golden Paths

**Status:** Canonical execution map

## Purpose

Convert the repository from a growing body of insight into a sequence of compounding, truth-aligned execution paths.

A golden path is not merely an interesting topic. It is a dependency-aware sequence in which each completed step unlocks multiple downstream capabilities.

## Golden Path 1 — Stabilize the Islamic authority stack

Sequence:

`Qur'an -> Sunnah -> hadith -> sira -> tafsir -> fiqh -> institutional practice -> empire -> local custom -> modern interpretation`

Why first:

Nearly every major controversy collapses source layers. Until this stack is mapped, debates over Aisha, slavery, apostasy, warfare, women, governance, and punishment remain structurally confused.

Outputs:

- source-layer ontology;
- authority rules;
- confidence classes;
- contradiction-resolution protocol;
- public claim language for “Islam says”.

## Golden Path 2 — Build the Truth-Closure Engine

Sequence:

`raw claim -> atomization -> source lineage -> definitions -> strongest cases -> causal graph -> breaker evidence -> provisional verdict -> public compression -> supersession trail`

Why second:

Propaganda immunity prevents false belief. Truth closure determines what should replace it.

Outputs:

- claim cards;
- source ledgers;
- versioned verdicts;
- prohibited-overclaim lists;
- correction history.

## Golden Path 3 — Stress-test the system on flagship controversies

Run the engine on:

1. Aisha age-report provenance;
2. Islam and slavery;
3. Muhammad and the “warlord” label;
4. early Islamic warfare and civilian protection;
5. Qur'an 5:32;
6. Banu Qurayza.

Why these:

Together they test source criticism, definition discipline, moral anachronism, blueprint versus implementation, legal architecture, hostile framing, and public communication.

## Golden Path 5 — Test inherited moral capital empirically

Core question:

Can secular societies regenerate costly moral behavior, trust, community, restraint, and intergenerational obligation without relying on inherited religious capital?

Required design:

- longitudinal rather than cross-sectional data;
- controls for wealth, education, state capacity, inequality, homogeneity, land distribution, family structure, and institutional quality;
- falsifiers for both religious and secular explanations;
- mechanism tracing, not simple correlation.

## Golden Path 6 — Map moral innovation versus moral completion

For each major moral domain, separate:

- baseline condition;
- revealed or philosophical innovation;
- institutional translation;
- historical compliance;
- unresolved injustice;
- later completion or regression.

Domains:

- slavery;
- civilian protection;
- women;
- race and ethnicity;
- poverty;
- animals;
- religious minorities;
- political accountability.

Canonical verdict form:

> This was a major moral advance relative to its baseline, but it did not complete the moral trajectory it opened.
